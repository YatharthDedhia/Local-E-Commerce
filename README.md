# Development guide
## **Step 1:** Clone the repo

    git clone https://github.com/YatharthDedhia/Online-Test-Platform.git

## **Step 2:** Open *frontend* branch
    
    git checkout frontend

## **Step 3:** Install required **npm modules**

    npm i

## **Step 4:** Start the local server

    npm start
